1.4 特殊矩陣

  • 元素全為0矩陣： np.zeros(<大小>, <資料型別>)
  • 元素全為1矩陣： np.ones(<大小>, <資料型別>) # 矩陣大小以 tuple 指定
  • 單位矩陣(Identity)： np.eye(<m>, <資料型別>) # eye 與 I 同音
  • 反矩陣(Inverse)： np.linalg.inv(<矩陣>)  # 行列式不可為0 
 1: import numpy as np
 2: 
 3: print('np.zeros:\n', np.zeros((3,5), dtype='int'))
 4: print('np.ones:\n', np.ones((3,5), dtype='int'))
 5: print('np.eyes:\n', np.eye(4, dtype='int'))
 6: x = np.random.randint( 0, 10, size=[3, 3])
 7: print('x:\n', x)
 8: xt = x.T # 轉置
 9: print('xt:\n', xt)
10: mat = xt.dot(x)
11: print('mat:\n', mat)
12: print('np.linalg.inv(mat):\n', np.linalg.inv(mat))
13:

1.5 ndarray屬性

  • dtype：ndarray每個元素的資料型別 (可於創建矩陣時同時指定)
    • int, float, bool, D(複數)
  • ndim：ndarray的維度
  • shape：ndarray的大小，即矩陣 m*n ( m 列 n 行 )
  • size：ndarray的元素個數
  • itemsize：ndarray中每個元素記憶體大小(單位byte)
1: import numpy as np
2: 
3: np1 = np.arange(12, dtype='D').reshape(3, 2, 2)
4: print(np1)
5: print('dtype=', np1.dtype)
6: print('shape=', np1.shape)
7: print('size=', np1.size)
8: print('itemsize=', np1.itemsize)
9: print('nbytes=', np1.nbytes)

1.6 矩陣形狀轉換

  • reshape(m, n, …)：不改變原矩陣形狀
  • resize(m, n, …)：和 reshape 類似，但會改變原矩陣形狀
  • ravel( ) / flatten( )：將多維矩陣轉換成一維矩陣，原矩陣不變
  • shape = (m, n, …)：用 tuple 來指定矩陣大小，會改變原矩陣

1.9 Numpy常用統計函數

  • np.sum(<np>)：回傳和
  • np.mean(<np>)：回傳平均值
  • np.max(<np>)：回傳最大值
  • np.min(<np>)：回傳最小值
  • np.ptp(<np>)：回傳最大值減最小值
  • np.std(<np>)：回傳標準差
  • np.var(<np>)：回傳變異數
  • np.cumsum(<np>)：回傳累加值
  • np.cumprod(<np>)：回傳累積值 註：使用時需要指定axis軸的方向，若不指定，預設統計整個numpy

7.2.2 實作: 取得HTML資料

  1. 第一版
    1: # 抓取PTT電影版header
2: import urllib.request as req
3: url = "https://www.ptt.cc/bbs/Stock/index.html"
4: 
5: with req.urlopen(url) as response:
6:     data = response.read().decode("utf-8")
7: print(data)

    urllib.error.HTTPError: HTTP Error 403: Forbidden

    被拒絕原因：這隻程式的行為不像一般使用者，被網站伺服器拒絕。403 Forbidden 是HTTP協議中的一個HTTP狀態碼（Status Code）。可以簡單的理解為沒有權限訪問此站，服務器收到請求但拒絕提供服務1。

  2. 第二版:加入header
     1: import urllib.request as req

 5:     "User-Agent":"Mozilla/5.0 (Macintosh; Intel Mac OS X 10.14; rv:72.0) Gecko/20100101 Firefox/72.0"
 6: })
 7: 
 8: with req.urlopen(request) as response:
 9:     data = response.read().decode("utf-8")
10: print('所取得的資料型態：',type(data))
